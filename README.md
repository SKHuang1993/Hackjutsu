# Hackjutsu

Hey, feel free to visit my blog:)

`Jutsu` means `skills` in Japanese, for example `Kenjutsu` means `skills of swords`. Inspired by that, I make the word `Hackjutsu` for `skills of hacking` here. Have fun~
